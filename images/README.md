## 📝 截图规范

### 技术要求
- **分辨率**：建议1920x1080或更高
- **格式**：PNG格式（支持透明背景）
- **文件大小**：单张不超过500KB
- **清晰度**：确保文字清晰可读

### 命名规范
- **格式**：`{模块}-{功能}-{序号}.png`
- **示例**：`task-batch-01.png`、`orders-list-01.png`
- **特殊标记**：使用`-overview`表示模块概览

### 内容要求
- **完整页面**：截取完整的页面内容
- **关键信息**：突出显示重要的操作按钮和功能区域
- **数据示例**：使用真实的示例数据（可脱敏处理）
- **状态标识**：显示不同的页面状态（加载中、成功、错误等）
