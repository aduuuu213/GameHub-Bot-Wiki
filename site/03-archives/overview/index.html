商品管理总览¶

功能简介¶

商品管理模块是系统的核心业务模块，负责管理游戏商品的全生命周期。从游戏号的登记管理，到商品档案的建立，再到商品的上架销售，系统提供完整的商品管理解决方案。

主要功能¶

游戏号管理¶

  • 游戏号登记：录入和管理各种游戏账号信息
  • 账号分类：按游戏类型、等级、价值等维度分类管理
  • 状态跟踪：实时跟踪游戏号的使用状态和可用性
  • 批量操作：支持批量导入、导出、修改游戏号信息

商品档案¶

  • 档案建立：为每个游戏号建立详细的商品档案
  • 信息维护：维护商品的基本信息、价格、描述等
  • 版本管理：跟踪商品信息的变更历史
  • 模板管理：使用模板快速创建相似商品档案

在售商品¶

  • 上架管理：管理商品的上架状态和销售信息
  • 价格策略：设置动态价格和促销策略
  • 库存监控：实时监控商品库存和销售状态
  • 销售统计：统计商品的销售数据和表现

专用上架¶

  • 特殊商品：管理需要特殊处理流程的商品
  • 定制服务：为特定客户提供定制化商品服务
  • 优先级管理：设置专用商品的优先级和处理规则
  • 专属渠道：为专用商品配置专属销售渠道

超期设置¶

  • 时间管理：设置商品的有效期和超期时间
  • 自动处理：配置超期商品的自动处理规则
  • 提醒机制：设置超期提醒和通知机制
  • 清理策略：制定过期商品的清理和归档策略

使用场景¶

游戏商品销售¶

  • 管理各类游戏账号的销售
  • 处理游戏道具和虚拟物品交易
  • 提供游戏代练和陪玩服务

商品生命周期管理¶

  • 从商品录入到销售完成的全流程管理
  • 商品状态变更和流程跟踪
  • 商品质量控制和售后服务

库存和价格管理¶

  • 实时库存监控和预警
  • 动态价格调整和促销管理
  • 销售数据分析和优化建议

权限要求¶

  • 查看权限：所有用户都可以查看商品信息
  • 编辑权限：需要"商品管理"权限才能修改商品信息
  • 上架权限：需要"商品上架"权限才能上架商品
  • 管理权限：需要"商品管理"权限才能删除商品
  • 系统权限：需要"系统管理"权限才能修改全局设置

子模块导航¶
