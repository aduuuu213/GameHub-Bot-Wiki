# 登录与注册

## 登录系统

### 访问登录页面
1. 在浏览器中输入系统地址
2. 系统会自动跳转到登录页面
3. 如未自动跳转，点击"登录"按钮

![登录页面截图](../images/login-page.png)

### 登录步骤
1. **输入用户名**：输入管理员提供的用户名
2. **输入密码**：输入初始密码或已设置的密码
3. **点击登录**：点击"登录"按钮
4. **验证通过**：系统验证成功后进入主界面

![登录表单截图](../images/login-form.png)

### 忘记密码
1. 在登录页面点击"忘记密码"链接
2. 输入注册邮箱地址
3. 系统发送重置密码邮件
4. 按照邮件指引重置密码

![忘记密码页面截图](../images/forgot-password.png)

## 用户注册

### 注册流程
1. **访问注册页面**：点击登录页面的"注册"按钮
2. **填写注册信息**：
   - 用户名（必填）
   - 邮箱地址（必填）
   - 手机号码（必填）
   - 密码（必填）
   - 确认密码（必填）
3. **提交注册**：点击"注册"按钮提交信息
4. **等待审核**：管理员审核通过后账号生效

![注册页面截图](../images/register-page.png)

### 账号激活
1. 注册成功后，系统会发送激活邮件
2. 点击邮件中的激活链接
3. 账号激活成功，可以正常登录

## 安全设置

### 修改密码
1. 登录系统后，点击右上角用户头像
2. 选择"个人设置"
3. 在"安全设置"中点击"修改密码"
4. 输入原密码和新密码
5. 确认修改

![修改密码页面截图](../images/change-password.png)

### 会话超时
- **自动超时**：系统会在30分钟无操作后自动登出
- **手动登出**：点击右上角"退出登录"
- **会话续期**：在超时前进行任何操作可自动续期

## 常见问题

### 无法登录
**可能原因及解决方案：**
- 用户名或密码错误 → 检查输入是否正确
- 账号被锁定 → 联系管理员解锁
- 网络连接问题 → 检查网络连接
- 浏览器兼容性 → 使用推荐浏览器

### 账号被锁定
**解锁方法：**
1. 联系系统管理员
2. 提供用户名和注册邮箱
3. 管理员核实身份后解锁账号
4. 重新设置密码
